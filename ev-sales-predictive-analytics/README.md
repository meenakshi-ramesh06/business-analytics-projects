# EV Sales Forecasting Using Predictive Analytics

## Business Objective

This project analyzes factors influencing electric vehicle (EV) sales and builds predictive models to forecast future demand patterns.

## Methods Used

* Descriptive Statistics
* Correlation Analysis
* Multiple Linear Regression
* OLS Regression
* Time Series Forecasting

## Forecasting Models

* Moving Average (MA)
* Simple Exponential Smoothing (SES)
* Holt's Linear Trend Model

## Key Insights

Battery capacity shows a strong positive relationship with EV sales, while discount percentage has minimal influence on consumer demand.

## Outcome

Trend-based forecasting methods such as Holt’s model provide more realistic projections for EV adoption compared to simple smoothing techniques.

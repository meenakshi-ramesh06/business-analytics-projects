# Customer Churn Prediction Using Logistic Regression

## Business Objective

The objective of this project is to predict which customers are likely to churn based on service usage and account characteristics. Predicting churn helps businesses implement targeted retention strategies.

## Dataset

IBM Telco Customer Churn Dataset

## Methods Used

* Correlation Analysis
* Logistic Regression
* Train-Test Split
* Confusion Matrix
* ROC Curve

## Key Insights

Customers with shorter tenure and higher monthly charges show a higher probability of churn. Predictive models can help businesses identify at-risk customers and reduce customer attrition.

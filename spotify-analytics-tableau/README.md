## Project Overview

Designed an interactive Tableau dashboard to analyze Spotify music data and uncover trends in artists, albums, genres, and track popularity. The dashboard provides an intuitive way to explore music insights using dynamic visualizations.

## Objectives

* Analyze music streaming trends.
* Compare artist and album performance.
* Explore genre distribution and track popularity.
* Present insights through interactive dashboards.

## Tools & Technologies

* Tableau
* CSV Dataset

## Dashboard Highlights

* Interactive dashboard with filters
* Artist performance analysis
* Album and genre insights
* Track popularity visualization
* Dynamic charts and KPIs

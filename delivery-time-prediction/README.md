# Delivery Time Prediction Using Machine Learning

## Business Objective

This project analyzes operational factors affecting delivery time and builds predictive models to estimate delivery duration. Understanding these factors can help logistics platforms improve delivery efficiency.

## Dataset

Food Delivery Time Prediction Dataset (Kaggle)

## Methods Used

* Correlation Analysis
* Linear Regression
* Decision Tree Regression
* Model Evaluation

## Key Insights

Delivery time is influenced by factors such as distance, preparation time, traffic conditions and courier experience. Predictive models can help logistics companies optimize delivery planning.

# Business Analytics Projects Portfolio

This repository contains a collection of business analytics projects developed as part of my coursework in the **BCom (Hons) Business Analytics program at MAHE Bengaluru**.

The projects demonstrate applications of **data analysis, predictive modeling, business intelligence, and AI-driven business insights** across multiple business domains.

---

# Projects Included

## 1. Gen Z Investor Behaviour & Brokerage Market Analysis

Power BI dashboard analyzing brokerage pricing strategies, market share, and app experience of discount and traditional brokers in India.

Key focus areas:

* Market share comparison
* AMC and pricing models
* Gen Z investor trends
* App experience and adoption

---

## 2. EV Sales Forecasting Using Predictive Analytics

Predictive analytics project analyzing electric vehicle sales patterns using regression and time-series forecasting models.

Methods used:

* Correlation analysis
* Multiple linear regression
* OLS regression
* Moving average forecasting
* Simple exponential smoothing
* Holt’s trend model

---

## 3. AI-Driven Customer Analytics for Tira Beauty

AI-based business analytics case study exploring how artificial intelligence can enhance customer engagement and personalization in the beauty retail industry.

Applications explored:

* AI chatbot concept
* Product recommendation system
* Sentiment analysis
* Feature importance analysis
* Lead scoring model

---

## 4. Customer Churn Prediction

Machine learning project predicting customer churn using logistic regression and classification techniques.

Methods used:

* Logistic regression
* Train-test split
* Confusion matrix
* ROC curve
* Model evaluation

---

## 5. Retail Customer Segmentation (RFM Analysis)

Customer analytics project segmenting customers based on purchasing behaviour using Recency, Frequency, and Monetary value analysis.

Applications:

* Customer segmentation
* Identification of high-value customers
* Marketing strategy insights

---

## 6. Delivery Time Prediction Using Machine Learning

Operations analytics project predicting delivery duration based on operational and environmental factors.

Methods used:

* Correlation analysis
* Linear regression
* Decision tree regression
* Model evaluation

## Project Overview

Developed an interactive Power BI dashboard to analyze Electric Vehicle (EV) adoption trends using real-world data. The dashboard enables users to explore vehicle distribution, manufacturer performance, model insights, and key adoption metrics through interactive visualizations.

## Objectives

* Analyze EV adoption patterns.
* Identify leading manufacturers and vehicle models.
* Present business insights using interactive dashboards.
* Support data-driven decision-making through visualization.

## Tools & Technologies

* Microsoft Power BI
* Microsoft Excel / CSV
* Data Cleaning & Transformation

## Dashboard Highlights

* Interactive filters and slicers
* EV population overview
* Manufacturer-wise analysis
* Vehicle type distribution
* Model-wise insights
* KPI cards and visual summaries

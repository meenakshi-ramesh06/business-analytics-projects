# Retail Customer Segmentation Using RFM Analysis

## Business Objective

The objective of this project is to segment customers based on their purchasing behaviour using Recency, Frequency and Monetary (RFM) analysis.

## Dataset

Online Retail Dataset (UCI Machine Learning Repository)

## Methods Used

* RFM Analysis
* Customer Segmentation
* Exploratory Data Analysis
* Sales Pattern Analysis

## Key Insights

Customers with high purchase frequency and high monetary value represent the most valuable segment. Customers with high recency scores may be at risk of churn.
